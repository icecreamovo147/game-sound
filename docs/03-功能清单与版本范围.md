# 03-功能清单与版本范围

## 1. 功能总览

GameSound TUI 的完整功能分为以下模块：

1. 音频库管理；
2. 分类与音效方案管理；
3. TUI 交互界面；
4. 播放控制；
5. 全局快捷键；
6. 麦克风采集；
7. 音效解码；
8. 实时混音；
9. 虚拟音频输出；
10. 本地监听；
11. 配置持久化；
12. 日志与诊断；
13. 导入导出；
14. 高级音频处理；
15. 后续 GUI 与虚拟驱动扩展。

## 2. 音频库管理功能

| 功能 | 说明 | 完整版本要求 |
|---|---|---|
| 添加音频 | 导入本地音频文件 | 支持单文件、多文件、文件夹扫描 |
| 删除音频 | 从库中移除音频 | 支持保留源文件或删除复制文件 |
| 编辑音频 | 修改名称、分类、音量等 | 支持弹窗编辑 |
| 搜索音频 | 根据关键词搜索 | 支持名称、标签、快捷键搜索 |
| 收藏音频 | 将常用音效标记收藏 | 收藏列表单独展示 |
| 排序音频 | 调整显示顺序 | 支持手动排序和自动排序 |
| 播放统计 | 记录播放次数 | 支持按最近使用排序 |

## 3. 分类与方案功能

| 功能 | 说明 |
|---|---|
| 分类创建 | 用户可创建自定义分类 |
| 分类重命名 | 修改分类名称 |
| 分类删除 | 删除分类时可迁移音效到默认分类 |
| 分类排序 | 调整分类显示顺序 |
| 音效方案 | 支持多套音效库和快捷键配置 |
| 方案切换 | 可在不同游戏场景间切换 |
| 方案导出 | 将方案打包为配置文件 |
| 方案导入 | 导入其他用户或备份方案 |

## 4. 播放控制功能

| 功能 | 说明 |
|---|---|
| 播放音效 | 通过 TUI 或全局快捷键播放 |
| 停止音效 | 停止指定音效 |
| 停止全部 | 立即停止所有音效 |
| 暂停/恢复 | 支持正在播放音效暂停与恢复 |
| 循环播放 | 支持 BGM 类音效循环 |
| 播放进度 | 显示播放状态和进度 |
| 多路播放 | 支持多个音效同时播放 |
| 队列播放 | 支持音效排队播放 |
| 打断播放 | 新音效打断当前音效 |
| 同组互斥 | 同分类内音效互斥播放 |

## 5. 音频处理功能

| 功能 | 说明 |
|---|---|
| 麦克风采集 | 从真实麦克风读取音频流 |
| 音效解码 | 解码 MP3、WAV、M4A 等文件 |
| 采样率转换 | 统一不同设备和文件采样率 |
| 声道转换 | 单声道、双声道转换 |
| 实时混音 | 麦克风与音效混合 |
| 限幅 | 防止混音后爆音 |
| Ducking | 音效播放时自动压低麦克风 |
| 噪声门 | 小音量背景噪声自动降低 |
| 电平统计 | 显示麦克风和输出电平 |
| 本地监听 | 将音效或混音输出到耳机 |

## 6. 设备管理功能

| 功能 | 说明 |
|---|---|
| 输入设备枚举 | 显示所有麦克风设备 |
| 输出设备枚举 | 显示所有输出设备 |
| 虚拟设备识别 | 标记可能的虚拟声卡 |
| 设备切换 | 运行中切换设备 |
| 设备测试 | 播放测试音或显示电平 |
| 设备失效处理 | 断开后提示用户重新选择 |
| 默认设备恢复 | 配置丢失时回退到默认设备 |

## 7. TUI 功能

| 功能 | 说明 |
|---|---|
| 主界面 | 展示音效列表、分类、设备、音量 |
| 配置页 | 展示并修改设备和全局配置 |
| 添加弹窗 | 添加新音效 |
| 编辑弹窗 | 编辑音效信息 |
| 快捷键绑定弹窗 | 监听并绑定快捷键 |
| 设备选择弹窗 | 切换麦克风、输出、监听设备 |
| 帮助页 | 展示操作快捷键和说明 |
| 日志页 | 查看运行日志 |
| 搜索模式 | 输入关键词筛选音效 |

## 8. 全局快捷键功能

| 功能 | 说明 |
|---|---|
| 音效快捷键 | 每个音效可绑定一个或多个快捷键 |
| 停止快捷键 | 一键停止所有音效 |
| 静音快捷键 | 一键静音麦克风或音效 |
| 方案切换快捷键 | 快速切换音效方案 |
| 音量快捷键 | 调整麦克风、音效和监听音量 |
| 冲突检测 | 检查快捷键重复和注册失败 |
| 运行中重载 | 修改快捷键后重新注册 |

## 9. 存储与配置功能

| 功能 | 说明 |
|---|---|
| SQLite 数据库 | 保存音效、分类、方案、统计信息 |
| TOML 配置 | 保存用户偏好和设备选择 |
| 自动保存 | 配置修改后自动保存 |
| 手动保存 | 用户显式保存 |
| 配置备份 | 定期备份配置 |
| 配置恢复 | 从备份恢复 |
| 配置迁移 | 版本升级时迁移数据结构 |

## 10. 完整版本实现原则

文档以完整功能为目标，但开发时可按优先级推进。完整版本最终应具备稳定的音频混音能力、完善的 TUI 操作体验、可靠的设备管理、丰富的音频库功能和可扩展的架构。
