# GameSound TUI 开发文档集

本压缩包是一套面向完整版本实现的开发文档，项目暂定名为 **GameSound TUI**。

## 项目定位

GameSound TUI 是一款运行在终端中的游戏语音音效板工具。用户可以导入自定义音频、设置全局快捷键，在游戏或语音聊天过程中快速触发播放。软件将真实麦克风输入与音效音频进行实时混音，并输出到虚拟麦克风设备，使 Discord、QQ、微信语音、KOOK、游戏内语音等第三方语音软件能够同时采集用户说话声和音效声。

## 推荐技术栈

- 语言：Rust
- TUI：Ratatui + Crossterm
- 音频 I/O：cpal
- 音频解码：Symphonia，后续可接入 FFmpeg
- 配置：serde + toml
- 数据库：SQLite / rusqlite
- 全局快捷键：global-hotkey / rdev
- 日志：tracing
- 异步与事件：tokio / crossbeam-channel
- 虚拟音频设备：Windows 使用 VB-CABLE，macOS 使用 BlackHole；后期可自研虚拟麦克风驱动
