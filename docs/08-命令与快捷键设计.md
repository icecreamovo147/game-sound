# 08-命令与快捷键设计

## 1. 快捷键分类

系统存在两类快捷键：

1. TUI 操作快捷键：仅在终端界面聚焦时生效；
2. 全局快捷键：程序后台运行时生效，即使用户正在游戏中也能触发。

两类快捷键必须分离管理，避免冲突。

## 2. TUI 操作快捷键

### 2.1 全局页面操作

| 按键 | 功能 |
|---|---|
| Q | 退出程序 |
| ? | 打开帮助页面 |
| Tab | 切换焦点区域 |
| Esc | 返回或关闭弹窗 |
| C | 打开配置页面 |
| S | 停止所有音效 |
| / | 搜索音效 |
| R | 刷新状态或设备 |

### 2.2 音效列表操作

| 按键 | 功能 |
|---|---|
| ↑ / ↓ | 选择音效 |
| Enter | 播放选中音效 |
| Space | 停止选中音效 |
| A | 添加音效 |
| E | 编辑音效 |
| D | 删除音效 |
| B | 绑定快捷键 |
| + | 增加选中音效音量 |
| - | 降低选中音效音量 |

### 2.3 设备操作

| 按键 | 功能 |
|---|---|
| M | 选择麦克风输入 |
| O | 选择虚拟输出设备 |
| L | 选择监听设备 |
| T | 播放测试音 |

## 3. 全局快捷键

### 3.1 音效播放快捷键

每个音效可绑定一个或多个全局快捷键。

示例：

| 音效 | 快捷键 |
|---|---|
| 哈哈哈 | Ctrl + 1 |
| 芭比Q | Ctrl + 2 |
| 登场BGM | Ctrl + 3 |

### 3.2 控制类快捷键

| 功能 | 默认建议 |
|---|---|
| 停止全部音效 | Ctrl + Alt + S |
| 静音麦克风 | Ctrl + Alt + M |
| 静音音效 | Ctrl + Alt + X |
| 开关本地监听 | Ctrl + Alt + L |
| 音效音量增加 | Ctrl + Alt + Up |
| 音效音量降低 | Ctrl + Alt + Down |

## 4. 快捷键绑定流程

```text
用户选择音效
→ 按 B 打开绑定弹窗
→ 系统进入快捷键捕获模式
→ 用户按下目标组合键
→ 系统检测冲突
→ 注册全局快捷键
→ 保存绑定
→ 更新音效列表
```

## 5. 冲突检测

需要检测：

- 是否已经被其他音效使用；
- 是否被控制类快捷键使用；
- 是否注册失败；
- 是否与 TUI 操作键产生歧义；
- 是否为空或无效组合。

处理策略：

- 默认阻止重复绑定；
- 支持用户确认后覆盖；
- 注册失败时恢复原快捷键。

## 6. 命令面板设计

完整版本可支持命令面板，按 `:` 或 `P` 打开。

命令示例：

```text
:add ./haha.mp3
:play 1
:stop-all
:set mic in-2
:set output out-3
:profile switch cs2
:export profile default
```

命令面板适合高级用户和脚本化操作。

## 7. 外部命令支持

除 TUI 外，可保留少量外部命令：

```bash
gamesound --help
gamesound --version
gamesound doctor
gamesound reset-config
gamesound import <file>
gamesound export <profile>
```

这些命令用于诊断、导入导出和恢复配置。
